# 环境检查与依赖安装脚本设计

## 概述

为 IMGR 项目添加环境检查和依赖安装脚本，帮助新用户在全新环境中快速完成环境验证和依赖安装。

## 脚本结构

```
scripts/
├── check_env.sh      # 环境检查脚本
├── install_deps.sh   # 依赖安装脚本
```

## check_env.sh — 环境检查

### 检查项目

| 检查项 | 要求 | 命令 |
|--------|------|------|
| Python | 3.11+ | `python3 --version` |
| Node.js | 18+ | `node --version` |
| npm | 9+ | `npm --version` |
| pip | 可用 | `python3 -m pip --version` |

### 输出格式

**满足要求时：**
```
========================================
  IMGR 环境检查
========================================
✓ Python 3.11.8 (要求 3.11+)
✓ Node.js 20.11.0 (要求 18+)
✓ npm 10.2.4 (要求 9+)
✓ pip 可用
========================================
✓ 环境检查通过
```

**不满足时：**
```
========================================
  IMGR 环境检查
========================================
✗ Python 3.10.1 (需要 3.11+)
✗ 未找到 Node.js (需要 18+)
========================================
✗ 环境检查未通过，请先安装所需环境
```

### 退出码

- 全部通过：返回 0
- 任何项不满足：返回非 0

## install_deps.sh — 依赖安装

### 执行步骤

1. **环境检查** — 调用 `check_env.sh`，不满足则退出
2. **后端依赖** — `pip install -r backend/requirements.txt`
3. **Playwright 浏览器** — `playwright install chromium`
4. **前端依赖** — `cd frontend && npm install`

### 输出格式

```
========================================
  IMGR 依赖安装
========================================
[1/4] 检查环境...
      ✓ 环境检查通过

[2/4] 安装后端依赖...
      ✓ 安装完成

[3/4] 安装 Playwright 浏览器...
      ✓ 安装完成

[4/4] 安装前端依赖...
      ✓ 安装完成

========================================
✓ 所有依赖安装完成
```

### 错误处理

- 每步骤检查返回码，失败时立即退出
- 显示具体失败的步骤和错误信息

## 关键设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 脚本位置 | `scripts/` 子目录 | 与启动脚本分离，职责清晰 |
| 依赖关系 | install_deps 调用 check_env | 避免代码重复 |
| Playwright | 仅安装 chromium | 满足项目需求，减少安装时间 |
| 输出风格 | 彩色符号 + 进度编号 | 便于用户了解进度和状态 |
